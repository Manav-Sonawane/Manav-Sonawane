- Reimagined PlayStation's digital platform as a social-first gaming ecosystem in 5 hours
- Dark immersive UI with collapsible sidebar, game carousel, SVG stats, achievements, and group chat
- Fully deployed on Vercel with smooth animations throughout

### 🔗 [Reclaim Frontend](https://github.com/Manav-Sonawane/reclaim-frontend)
**Frontend Project** | Next.js, React, Tailwind CSS

- Clean, responsive UI with component-based architecture

**Movie Ticket Booking Platform** | Django, PostgreSQL, Bootstrap

- Payment gateway integration and admin analytics dashboard

**AI-Powered Resume Analyzer** | Python, Flask, Gemini API, NLP
- Resume parser using NLP + OCR (PyTesseract) for PDF extraction
- Personalized career recommendations and skill gap analysis

### 👤 [Facial Recognition Attendance](https://github.com/Manav-Sonawane/Facial-Attendence)
**Smart India Hackathon Prototype** | Python, Flask, DeepFace, MySQL

- PyQt admin interface for attendance management

### 💻 [CloudCode](https://github.com/Manav-Sonawane/CodeCloud)
